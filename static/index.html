Ferret API

API для получения хорьков

Для чего?

Изначально - для бота Sculk, а именно для команды /хорёк. Но можно использовать и не только для этого!

Как получить картинку?

HTTP GET запрос на /image

GET https://ferretapi.ru/image

А если в JSON?

Относительную ссылку на картинку хорька можно получить через /json

GET https://ferretapi.ru/json
{
    "ferret": "/ferret-image/00000000-0000-4000-0000-000000000000.jpg"
}

Остается добавить https://ferretapi.ru в начало, чтобы получить картинку:

https://ferretapi.ru/ferret-image/00000000-0000-4000-0000-000000000000.jpg
